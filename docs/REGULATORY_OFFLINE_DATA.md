# Офлайн «база» регуляторних посилань

## Що це **не** є

Повні національні або наднаціональні **бази норм** (EUR-Lex, eCFR, державні реєстри тощо) — це **мільйони** документів і постійні оновлення. Їх **не** вбудовують у мобільний застосунок як один завантажуваний дамп: розмір, ліцензії, ToS API та підтримка роблять це непрактичним.

## Що робить застосунок замість цього

1. **`CountryRegulatoryContext`** — уже містить **офіційні стартові посилання** (`quickLinks`) під обрану в профілі **країну** (юрисдикцію).
2. **`app/src/main/assets/regulatory_offline_index.json`** — **кураторський** шар:
   - **`sectorDefaults`** — типові портали на **сектор** (ключ як у `SectorKeys` / `UserProfile.sector`);
   - **`byCountry`** — додаткові URL під **країнний bucket** (`regulatoryCountryBucket`: `usa`, `ukraine`, `poland`, `germany`, `france`, `finland`, `netherlands`, `other`, `eu`);
   - **`nicheExtras`** — додаткові посилання за **`promptKey` ніші** (ті самі рядки, що в `UserProfile.niches` після онбордингу).

3. **`RegulatoryOfflineIndex.mergedOfficialLinks(...)`** збирає список у порядку: країна → сектор → країна+сектор → ніші з профілю (дедуплікація за URL).

Інтерфейс: **Profile**, **Settings**, **Glossary** показують об’єднаний список.

## Як розширювати

- Додавайте рядки в `regulatory_offline_index.json` (перевіряйте URL у браузері).
- Для нової **ніші** використовуйте **точний** `promptKey` з `NicheCatalog` (наприклад `chem_reach_registration`).
- Після великих змін має сенс підняти `"version"` у JSON для обліку (логіка застосунку поки не залежить від номера версії).

## Де брати «справжні» дані для серверної синхронізації (не в APK)

Приклади офіційних програмних інтерфейсів / порталів (для майбутнього backend, не для повного клонування в телефон):

- [EUR-Lex](https://eur-lex.europa.eu/) — пошук та API з обмеженнями;
- [data.europa.eu](https://data.europa.eu/) — відкриті набори ЄС;
- [FDA OpenData](https://open.fda.gov/) — обмежені набори по продуктах США;
- [ECHA](https://echa.europa.eu/), [EMA](https://www.ema.europa.eu/) — портали агентств.

Це **не** заміна юридичної консультації; критичні рішення завжди звіряйте з первинним джерелом.
